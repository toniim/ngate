🔐 Trust mkcert Root CA

    

      Download the root CA certificate, then install it on your device:
    

    

      
        ⬇ Download rootCA.pem
      
    

    

      
macOS:

      
sudo security add-trusted-cert -d -r trustRoot -k /Library/Keychains/System.keychain rootCA.pem

      

      
Windows (Admin PowerShell):

      
certutil -addstore -f "ROOT" rootCA.pem

      

      
Linux:

      
sudo cp rootCA.pem /usr/local/share/ca-certificates/mkcert-ca.crt && sudo update-ca-certificates

      

      
iOS:

      
Download rootCA.pem → Settings → Profile Downloaded → Install → Settings → General → About → Certificate Trust Settings → Enable

      

      
Android:

      
Download rootCA.pem → Settings → Security → Install from storage → CA certificate
